# CivicLens – AI Powered Civic Issue Reporting Platform 🏙️📸🤖

https://namma-oor-fix.vercel.app/

An AI-powered smart civic engagement platform that enables citizens to instantly report public issues like potholes, garbage dumps, water leakage, broken streetlights, traffic congestion, and more using image-based detection and geolocation tracking.

The platform bridges the gap between citizens and local authorities through real-time reporting, intelligent issue categorization, and interactive dashboards.

---

## 🚀 Features

### 📸 AI-Based Issue Detection

Users can upload photos of civic problems, and the system automatically analyzes and categorizes issues using AI.

### 📍 Real-Time Location Tracking

Automatically captures the user’s current location using Geolocation APIs for accurate issue reporting.

### 🧠 Smart Categorization

Detects categories such as:

* Potholes
* Garbage accumulation
* Water leakage
* Road damage
* Broken streetlights
* Illegal dumping
* Drainage overflow

### 🗺️ Interactive Issue Mapping

All reported issues are displayed on a live interactive map for better visibility and monitoring.

### 📊 Admin Dashboard

Authorities can:

* Monitor complaints
* Track issue status
* View analytics
* Prioritize high-risk zones
* Manage citizen reports efficiently

### 🔔 Real-Time Status Updates

Citizens receive updates when:

* Issue is verified
* Work begins
* Complaint gets resolved

### 🌙 Modern Cyberpunk UI

Futuristic dark-themed interface with neon gradients, smooth animations, and responsive layouts.

### 📱 Fully Responsive

Optimized for:

* Mobile devices
* Tablets
* Desktop browsers

# 💡 Example Use Case

Suppose a citizen notices:

* A large pothole
* Overflowing garbage
* Water leakage near a road

The user simply:

1. Opens CivicLens
2. Uploads a photo
3. AI detects the issue type
4. Location is auto-captured
5. Complaint gets submitted instantly

Local authorities can then view the issue on the dashboard and take action quickly.
